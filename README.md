# Meridian

Neutral, non-custodial infrastructure for creating ETFs and tokenized-equity baskets that work 24/7, with an honest price even when the market is closed.

## Start here

1. Entry point (must see): https://meridian-landing.up.railway.app
2. App production: https://meridianapp-production.up.railway.app
3. App-demo, no wallet: https://meridianapp-demo.up.railway.app
4. Playground: https://meridian-playground.up.railway.app

## Code

| Folder | What |
|---|---|
| [`backend/`](backend) | Off-chain service (Node, viem): signed price committee, market data, orchestration. |
| [`app/`](app) | Frontend (React, Vite, viem/wagmi): create a fund, enter and exit. |
| [`blockchain/`](blockchain) | Solidity contracts |
| [`gitbook/`](gitbook) | Protocol docs |
