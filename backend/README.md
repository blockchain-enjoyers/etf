# @meridian/backend

Node.js + TypeScript API. Web framework not chosen yet (Express/Fastify/NestJS — decide in the backend pass).

- Reads chain via `@meridian/contracts`.
- API DTO types are the single source of truth in `@meridian/sdk` (type-only imports here).
- `yarn workspace @meridian/backend dev` — watch-run the entrypoint.
